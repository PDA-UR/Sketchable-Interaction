Detailed Description

MapExposureString2_String2FunctionMap_Map class providing the interface for exposing a STL map to the python3 bindings (PySI) in a pythonic way which provides the functionality of a map which uses capabilities as keys to other maps which use event keywords, such as on_enter, on_continous, and on_leave for collision events, or another capbility for linking events as keys to the event function.
